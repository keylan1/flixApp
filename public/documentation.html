The flixApp is a web application that provides users with information on movies and lets them create favorite lists of movies to watch.

Business Logic URL HTTP Method Request body data format Response body data format
Get intro page / GET None Welcome to the flixApp!
Get documentation /documentation GET None HTML with documentation and request/response table
Get a list of movies /movies GET None JSON obj with list of movies
