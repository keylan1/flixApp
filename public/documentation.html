The flixApp is a web application that provides users with information on movies and lets them create favorite lists of movies to watch.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None JSON obj with list of movies
Get data bout a movie /movies/[name] GET None JSON obj about a single movie with ID, name, director (bio, birth, death), genre (name and description), description, image url, feateured.
Allow new users to register /users POST JSON with username, email, favorite movie Text message confirming registration with username \& email
Allow users to update their username /users/[userName] PUT None Text message indicating the username was updated
Allow users to add a movie to their favorites /users/[username]/[favorites] PUT None Text message indicating movie added by username
Allow users to delete a movie from favorites /users/[username]/[favorites] DELETE None Text message confirming a movie was deleted from favorites
Allow users to deregister via email /users/[userName]/[email] DELETE None Text confirming profile (username \& email) was deleted
